# Nano Letters 投稿前チェックリスト

## 0. 論文種別・全体方針
- [x] ジャーナルは **Nano Letters（ACS）** である
- [x] 論文種別は **Letter（Communication）** である
- [x] 「速報性・高い新規性」がカバーレターで説明できる

---

## 1. 文字数・構成制限（重要）

### 本文（Main Manuscript）
- [x] 本文の総語数が **3000 words 以下**（安全側）
- [x] Abstract が **150 words 以下**
- [x] 図（Scheme 含む）の数が **5 枚以下**（安全側）
- [x] Table は必要最小限（原則は Figure 中心）
- [x] セクション見出しを多用せず、**流れる文章構成**になっている
- [x] Introduction / Results / Discussion / Conclusion を分けすぎていない

---

## 2. 図・表（Figures & Tables）
- [x] すべての Figure が **本文中に埋め込まれている**
- [x] 各 Figure が本文中で必ず言及されている
- [x] Figure キャプションが **自己完結的**（SIを見なくても理解可能）
- [x] 軸ラベル・単位・スケールバーが明記されている
- [x] 解像度が十分（印刷耐性あり）
- [x] カラーバーの意味が明確
- [x] Figure 番号の不整合がない

---

## 3. Supporting Information（SI）
- [x] SI を **本文とは別ファイル**として用意している
- [x] SI ファイル名が分かりやすい（例：`Supporting_Information.pdf`）
- [x] 本文中に以下の記述がある  
  - [x] `Supporting Information: Experimental details, additional figures, ... (PDF)`
- [x] 本文の主張が **SI に依存していない**
- [x] SI の Figure / Table 番号が本文と混ざっていない（Figure S1 など）

---

## 4. TOC（Table of Contents）Graphic
- [x] TOC Graphic を用意している
- [x] サイズが **3.25 in × 1.75 in 以内**
- [x] 文字が小さすぎない
- [x] 1 枚で論文の「核」が分かる構成
- [x] 装飾過多になっていない

---

## 5. 参考文献（References）
- [x] 参考文献情報が **完全**（著者・タイトル・誌名・年・巻・ページ）
- [x] 最新の関連論文が適切に引用されている
- [x] 自己引用が過剰でない
- [x] 引用番号のズレがない

---

## 6. The cover letter should include the following:
- [ ] Manuscript title
- [ ] Corresponding author’s name and e-mail address
- [ ] 以下を明確に説明している
  - [ ] Statement on the manuscript’s significance
  - [ ] Comments on the manuscript’s novelty, impact, urgency
